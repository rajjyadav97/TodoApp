# TodoApp
A simple and responsive Todo App built using HTML, CSS, and JavaScript.

## Features
  - Add new tasks
  - Mark tasks as completed
  - Delete tasks
  - Input validation (empty tasks are not allowed)
  - Responsive UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## What I Learned
- DOM Manipulation
- Event Listeners
- Functions
- Arrays
- Input Validation
- Dynamic UI Updates
